# osrm-extensions-rs

Rust library for extending OSRM route responses with visual instructions and voice banners.

## Structure

- **`i18n`** - Internationalization support for voice instructions
- **`instructions`** - Visual instruction generation and formatting
- **`locales/`** - Locale data (e.g., number formatting rules)

## Usage

This library appends standardized JSON modules to route data from OSRM servers.
